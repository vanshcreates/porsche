# Porsche-Inspired Frontend Project

## Overview

This project is a frontend practice application built for learning and improving web development skills. It is inspired by the design and user experience of the Porsche website but was developed independently using my own HTML, CSS, and JavaScript code.

## Features

* Responsive layout
* Interactive image slider built with JavaScript
* Smooth animations and transitions
* Multiple custom-designed pages
* DOM manipulation and event handling
* Modern UI implementation

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Purpose

This project was created solely for educational and practice purposes to strengthen frontend development skills. It is not an official Porsche website and has no affiliation with Porsche AG.
